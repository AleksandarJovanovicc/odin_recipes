## Overview

The website features a main page that lists all available recipes. Each recipe has its own individual page, where you'll find everything you need to prepare the dish, including ingredients, step-by-step instructions, and images.

## Technologies Used

This website was created using only **HTML**. Key HTML elements utilized include:

- **Ordered and Unordered Lists**: For displaying lists of ingredients and steps in the recipes.
- **Paragraphs**: To provide descriptions and detailed instructions for each recipe.
- **Headings**: To structure the content and provide clarity for different sections of the page.
- **Images**: To visually represent the dishes and make the website more appealing.
- **Links**: For easy navigation between the main recipe page and individual recipe pages.

## Features

- **Main Page**: A clean and simple list of all available recipes, with clickable links to each recipe's individual page.
- **Recipe Pages**: Each recipe has its own dedicated page with ingredients, instructions, and a photo of the final dish.
- **Easy Navigation**: Links between the pages allow users to navigate easily through the recipe collection.

## How to Use

1. Open the website's main page to see the list of available recipes.
2. Click on any recipe link to go to the individual recipe page.
3. Follow the recipe instructions to cook up something delicious!

## Conclusion

This project was created as a simple showcase of using basic HTML to organize and display content.
